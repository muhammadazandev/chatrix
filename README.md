Chatrix is a modern, real-time chat web application built with React for the frontend and Express for the backend. It’s designed to provide a clean, scalable foundation for real-time messaging with support for multi-platform features.

Features (planned / implemented):

Real-time messaging (WebSocket support)
RESTful API for backend operations
Clean project structure (frontend + backend)
Environment variable management and proper .gitignore setup
Audio and video call support

Tech Stack:

Frontend: React, Tailwind CSS, Vite
Backend: Node.js, Express, CORS
Version Control: Git & GitHub
Environment Management: .env files for secrets

Purpose:
Chatrix is built as a learning project with a focus on clean architecture, best practices for Git workflow, and scalable design for future expansion features.
